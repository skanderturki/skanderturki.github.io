Features

Current offered features

Graphical Design

Design of the courses flow using a UML-like graphical interface.

Document Generation

Generate full curriculum documentation in Word, HTML, Text and SQL to update your database with the new curriculum.

Tailoring

We can tailor the tool for you to generate specific documents (excel, word, etc.) and to integrate your own business rules.

Many property views

To specify detailed descriptions of the curriculum, courses, modules, etc, the tool provides many property views.

Extensibility

The tool is based on eclipse and can be extended indefinetely.

Rich diagram edition tools

Zooming, image export, copy-paste, actions on multiple-selections, automatic rearrangement of diagram elements, show/hide elements, file explorer, multiple projects support, etc.

How to design a curriculum?

Simply create a new project, then create a new diagram. Now you can edit the diagram like any classical UML tool.

How to specify curriculum details?

A properties view allows to to specify your curriculum details in a very intuitive way.

How to specify course details?

A properties view allows to to specify the details of each course in a very intuitive way.

What can be automatically generated?

Out of the box, the tool allows to generate two word documents (program specification + all course specifications), an HTML webpage that contains a full documentation about the curriculum and a full SQL script that can be used to add the new curriculum to your relational database.
